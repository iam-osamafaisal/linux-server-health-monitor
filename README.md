# Linux Server Health Monitoring Script

This project monitors Linux server health including CPU, memory, disk usage, and uptime.

## Tools Used
- Linux
- Bash Shell
- Cron Job

## How to Run
1. chmod +x system_health.sh
2. ./system_health.sh

## Output
All system health data is saved in `health_report.log`.

## Use Case
Useful for Linux Administrators and Cloud Support Engineers to monitor server health.
